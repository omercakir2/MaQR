📌 Application Features

MaQR allows you to easily generate QR codes from any URL (even texts). Just enter a valid link and optionally customize the foreground and background colors

✅ Features:

  • 🧾 Generate a QR code from any URL
  • 🎨 Choose custom colors for foreground and background
  • 🔒 Validates the input and warns for empty or invalid URLs
  • 💾 Instantly download the generated QR code
  • 🖼️ Live preview of the QR code on the page

🚀 How to Use:

  1. Enter a URL
  2. Choose your preferred colors (optional)
  3. Click the “Submit” button
  4. Your QR code will be generated instantly!
